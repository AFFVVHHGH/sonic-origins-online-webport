How to Add Your Own Game:
To add a new game webport to this launcher interface, follow these two clean steps:

Step 1: Add the Selection Button
Open my index.html file, find the sidebar nav section, and paste a new button line right where you want it to appear. You will use a launcher button tag that triggers the bootGame function with your unique game ID and title.

Step 2: Add the Content Display Frame
Scroll down to the main viewport area and add a corresponding iframe container. Make sure the iframe ID perfectly matches the game ID you used in the button, and lock in your custom game URL link right inside the data source attribute.
